# Enterprise Multi-Branch Network - CCNA Project

## Technologies Used

* VLANs
* Inter-VLAN Routing
* OSPF Dynamic Routing
* WAN Serial Connection
* DHCP
* Multilayer Switches
* Cisco Packet Tracer

## Project Overview

This project simulates a real enterprise company with two branches connected through WAN links.

Each branch contains:

* Reception Department
* Sales Department
* Accounting Department
* Dedicated VLANs
* Dynamic routing using OSPF

## Features

* Full connectivity between branches
* Dynamic routing with OSPF
* Inter-VLAN communication
* DHCP configuration
* WAN connectivity

## Tools

* Cisco Packet Tracer
* Cisco IOS
